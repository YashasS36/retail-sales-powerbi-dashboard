# 📊 Retail Sales Analytics Dashboard - Power BI

## 🚀 Project Overview
This project is an interactive Retail Sales Analytics Dashboard built using Power BI.

The dashboard provides insights into:
- Total Sales
- Quantity Sold
- Product-wise Sales
- Customer-wise Sales
- Sales Trends Over Time

---

## 🛠 Tools & Technologies

- Excel

## 📈 Dashboard Features

- Interactive Filters
- Sales Trend Analysis
- Customer Distribution Analysis
- Product Performance Insights
